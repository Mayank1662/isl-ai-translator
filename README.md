# 🖐️ ISL AI Translator (Web + Real-Time)

A real-time Sign Language to Speech Translator using Computer Vision and Web Interface.

## 🚀 Features

- 🎥 Live camera gesture detection
- 🧠 AI-based hand tracking (MediaPipe)
- 🔊 Speech output (Text-to-Speech)
- ⚡ Real-time performance
- 📊 FPS + confidence display

---

## 🛠️ Tech Stack

- Python
- OpenCV
- MediaPipe
